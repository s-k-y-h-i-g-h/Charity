# Contributing

We welcome contributions from everyone. Here's how to help.

---

## What We Need

### Research
- **Peer-reviewed studies** on stack compounds
- **Human clinical trials** worth tracking
- **Regulatory updates** affecting access
- **Safety data** on compounds

### Content
- **Simplification** of complex research
- **Translations** to other languages
- **Accessibility** improvements
- **Error corrections**

### Technical
- **Site improvements** (HTML/CSS/JS)
- **Automation enhancements**
- **RSS/feed setup**
- **Search functionality**

---

## How to Contribute

### 1. Found a Study?
Open an issue with:
- Title and authors
- Link to paper
- Key findings
- Relevance to our mission

### 2. Spot an Error?
Open an issue with:
- What's wrong
- What should be there
- Source for correction

### 3. Want to Improve the Site?
Submit a pull request with:
- Clear description of changes
- Screenshots if UI changed
- Testing notes

---

## Guidelines

### Evidence Hierarchy
1. **Human clinical trials** (highest priority)
2. **Peer-reviewed studies**
3. **Reputable science news**
4. **Preclinical data** (with caveats)
5. **Anecdotal** (lowest priority)

### Tone
- **Direct, not hype**
- **Evidence-based, not promotional**
- **Accessible, not dumbed down**
- **Transparent about limitations**

### Safety
- **Never medical advice**
- **Always include disclaimers**
- **Flag controversies**
- **Verify before publishing**

---

## Review Process

1. **Submit** via issue or PR
2. **Review** by Ember & Sky
3. **Verify** with sources
4. **Publish** if validated
5. **Credit** contributors

---

## Code of Conduct

- **Respect** all contributors
- **Evidence over opinion**
- **No gatekeeping**
- **Accessibility first**
